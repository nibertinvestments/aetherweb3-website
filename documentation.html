Quick Start

1. Get Your API Key

Sign up for a free account to receive your API key instantly. No credit card required for the free tier.

Sign Up Free

2. Make Your First Request

curl -X POST https://api.aetherweb3.xyz/api/v1/ethereum \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getBalance",
    "params": ["0x742d35cc6634c0532925a3b8d3ac74e4dc3c8ef5", "latest"],
    "id": 1
  }'

3. Handle the Response

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1b1ae4d6e2ef500000"
}

WebSocket Streaming

Real-time Data Streaming

Connect to our WebSocket endpoints for real-time blockchain data:

const ws = new WebSocket('wss://api.aetherweb3.xyz/ws/v1/ethereum?auth=YOUR_API_KEY');

ws.onopen = function() {
  // Subscribe to new blocks
  ws.send(JSON.stringify({
    "jsonrpc": "2.0",
    "method": "eth_subscribe",
    "params": ["newHeads"],
    "id": 1
  }));
};

ws.onmessage = function(event) {
  const data = JSON.parse(event.data);
  console.log('New block:', data);
};

Server-Sent Events

Stream blockchain data with Server-Sent Events for web applications:

const eventSource = new EventSource('https://api.aetherweb3.xyz/api/v1/stream/ethereum?auth=YOUR_API_KEY');

eventSource.onmessage = function(event) {
  const blockData = JSON.parse(event.data);
  console.log('New block:', blockData);
};

eventSource.onerror = function(event) {
  console.error('Connection error:', event);
};
